## 🇨🇳 中文说明

### 💬 作者的话

这个插件最初只是我学习 Unreal Slate 时用来练手的小项目。

我平时习惯先用中文写下方法名、按钮文字或提示文本，再快速翻译成英文。JetBrains 系列 IDE（例如 PyCharm、Rider）中的 Translation 插件给了我一些灵感，于是我尝试在 Unreal Editor 中实现类似的快速翻译功能。

快速翻译本身可能算不上什么重要功能，但在开发过程中，我练习了 Slate UI、HTTP 请求、JSON 解析、插件设置以及第三方翻译 API 的接入，也算是这个插件的起点。🌱

本插件在开发过程中使用了 AI 辅助，代码由我自行编写、整合、调试与验证，主要用于个人练习、学习和技术交流。🤖✨

部分 Google 和 Microsoft 翻译代码参考了其他项目。其中 Google 翻译的实现相对简单，目前基本能够理解和维护；Microsoft 翻译部分的实现逻辑我暂时还没有完全掌握，因此现阶段不建议使用。
